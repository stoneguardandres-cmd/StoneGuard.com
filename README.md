# 🏗️ StoneGuard — Sitio Web Premium

Sitio web de alta conversión para StoneGuard, marca premium de selladores
nanotecnológicos y protección de superficies de lujo.

## 🎯 Características Principales

✅ **Diseño Premium Dark-Luxury**: Estética profesional con acentos rojos metálicos
✅ **Optimización de Conversión**: Múltiples CTAs estratégicamente ubicadas
✅ **Responsive Design**: Funcional en todos los dispositivos (mobile, tablet, desktop)
✅ **Performance**: Optimizado para Lighthouse, Core Web Vitals, PageSpeed Insights
✅ **SEO Ready**: Meta tags, Open Graph, estructuración semántica, sitemaps
✅ **Animaciones Suaves**: Framer Motion para transiciones elegantes
✅ **Componentes Reutilizables**: Arquitectura escalable y mantenible
✅ **TypeScript**: Type safety completo en todo el proyecto
✅ **Producción Ready**: Listo para desplegar en Vercel

## 🛠️ Stack Tecnológico

| Tecnología | Versión | Propósito |
|-----------|---------|-----------|
| **Next.js** | ^14.0.0 | Framework React moderno |
| **React** | ^18.2.0 | Librería UI |
| **TypeScript** | ^5.0.0 | Type safety |
| **Tailwind CSS** | ^3.3.0 | Estilos utility-first |
| **Framer Motion** | ^10.16.0 | Animaciones |
